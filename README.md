# Widget Meteo

Questo gadget implementa un componente React per presentare le condizioni meteorologiche adeguate ad una certa località. Esso contiene **tre viste di navigazione, scorribili per mezzo di swipe**, 

1. **Le condizioni meteorologiche correnti**
2. **Le previsioni del meteo sugli 5 prossimo ore**
3. **Le previsioni del meteo per i prossimi 5 giorni.**

La località viene data come parametro al componente e non è modificabile da parte del user.

## Tecnologia usata

- **JavaScript (ES6+)**
- **React.js**
- **TailWindCss**
- **Open-Meteo API e BigDataCloud API** (https://open-meteo.com/) per ottenere i dati sul meteo e (https://www.bigdatacloud.com/) per ottenere i dati del geolocalizzazione;
